# Cursorex Public Releases

Public release artifacts for Cursorex.

This repository is used as a download source for users who install the Chrome extension from the Chrome Web Store and need compatible local bridge binaries.

What is published here:
- Bridge release bundles built from the private source repository
- Native host and bridge runtime artifacts needed by local setup
- Versioned release assets for installation and rollback

Source code and CI live in the private repository. This repo is intentionally minimal and release-focused.
